# 🚀 StudyPath — Forje o seu Futuro

Plataforma de estudos moderna com foco em **gamificação + inteligência artificial**, criada para transformar a forma como os usuários aprendem.

🌐 **Acesse o projeto online:**

---

## 📌 Sobre o Projeto

O **StudyPath** é uma landing page interativa que apresenta o conceito de uma plataforma de estudos inteligente.

A proposta é simples:

> **Você foca em aprender, a IA cria o caminho.**

O sistema utiliza:

* 🧠 Inteligência Artificial (conceitual)
* 🎮 Gamificação (XP, progresso, evolução)
* 📊 Visual moderno com foco em experiência do usuário

---

## ✨ Funcionalidades

* 🔹 Interface moderna com **Glassmorphism**
* 🔹 Layout responsivo (Desktop + Mobile)
* 🔹 Sistema de **XP dinâmico**
* 🔹 Animações e efeitos visuais suaves
* 🔹 Integração com modelo 3D (`model-viewer`)
* 🔹 Navegação por âncoras (scroll suave)
* 🔹 Estrutura pronta para evolução em sistema real (SaaS)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox + Grid + Responsividade)
* JavaScript
* Google Fonts (Inter)
* Model Viewer (3D Web Component)
* Font Awesome (ícones)
